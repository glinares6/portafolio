    1-   
        npm install  --global gulp-cli
    Este comando lo tendras que  ejecutar usa sola vez.sirve para instalar gulp de forma global en el equipo para tenerlo disponible en todos los proyectos

    2- instalaciones que se han de generar en casa proyecto en las que querais incluir gulp
