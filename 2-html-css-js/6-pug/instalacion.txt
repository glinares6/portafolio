tener node en el sistema

iniciar un paquete ya existente
    npm install
    yarn



compilar pug en modo desarrollo
    pug --pretty  --watch pug1 --out  html1

compilar pug en otra carpeta en modo produccion
    pug --watch pug1  --out  html1

inicializar un proyecto pug en vite
    via npm 
        npm init @vitejs/app
    via yarn
        yarn create vite


lanzar el servidor
    via npm (previamente se debe hacer las instalción necesarias -webpack)
        npm run dev
    via yarn (ya viene integrado)
        yarn dev
    

renderizar el pug (en modo local-desarrollo)
    pug --pretty  --watch pug1 --out ./

rederizar sass (en modo local- desarrollo)
    sass --watch sass1:./
