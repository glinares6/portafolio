


devemos tener instalado npm y node en la pc

    npm init @vitejs/app
        le damos si (y)

    elegimos el marco en que vamos a desarrollar


luego instalamos las dependencias para que funcione vitejs
        npm install


para ejecutar vitejs
    npm run dev
                mas detalles en el package.json
                

el css lo compila de inmediato por que no recarga la pagina ya que actua en la misma.

    para usar las dependencias se debe instalar por ejemplo
        lodash
            npm install lodash-es

    podemos cambiar el file js a ts , considerar que vite no vendra  tipificado asi que tocara que le asignemos nosotros mismos.

    en el caso de dependencias de node,  vitejs ya lo trae incorporado en pequeños modulos dentro de un archivo(.vite)  que hace que el rendimoento sea más rapido
